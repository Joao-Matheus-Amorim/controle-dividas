# Escopo - FamilyFinance

## Declaracao

O FamilyFinance sera uma solucao personalizada para uma familia especifica, com painel web administrativo e app nativo Android/iOS planejado.

## Dentro do escopo

- Admin familiar.
- Usuarios familiares.
- Membros financeiros.
- Limites mensais.
- Gastos.
- Contas a pagar.
- Contas a receber.
- Bancos.
- Relatorios.
- Configuracoes.
- Permissoes por modulo.
- Permissoes de ver, criar, editar e excluir.

## Fora do escopo inicial

- Produto publico.
- Multiplas familias.
- Planos comerciais.
- Integracao bancaria automatica.
- Open Finance.
- Inteligencia artificial financeira.

## Produto atual

MVP web com dashboard, pessoas, gastos, contas, bancos, relatorios e configuracoes.

## Produto alvo

App nativo familiar para uso diario no celular, com painel web de apoio para o Admin.
